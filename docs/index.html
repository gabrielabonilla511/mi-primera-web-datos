Autora: Gabriela Bonilla

Este proyecto analiza publicaciones recientes del blog oficial de Python, con el objetivo de identificar patrones básicos en los títulos y la frecuencia de publicación del contenido.

Hallazgos principales

  • El blog de Python mantiene una publicación constante.
  • La mayoría de títulos contienen términos técnicos recurrentes.
  • El lenguaje utilizado es mayormente informativo y educativo.
